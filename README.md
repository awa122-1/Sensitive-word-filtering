# AmongUs敏感词过滤模组 

一个专为 Among Us 打造的绿色游戏环境增强模组。本模组在后台提供无痕聊天防护，同时支持完美的自定义主界面水印换肤功能。

---

## ✨ 核心功能

*   **🛡️ 绿色聊天防护**：自动扫描和过滤局内所有玩家的违规发言，一旦触发敏感词，消息将在本地直接静默蒸发，还你一片纯净的聊天屏。
*   **🎨 原生水印换肤**：无缝替换游戏主界面官方原版的模组标签（ModStamp），将其换成属于你自己的专属个性化图标，且文字和排版依然完美顺应官方原生样式。

---

## 💾 安装与使用方法

1.  确保你的游戏已经正确安装了 **BepInEx 6 (IL2CPP)** 或者是对应的稳定版模组加载框架。
2.  下载并解压本模组，将得到的 `tfumggl.dll` 文件放入游戏的插件目录中：
    ```text
    Among Us/BepInEx/plugins/
    ```
3.  **【配置你的专属图标】**：
    *   准备一张你喜欢的图片（建议为正方形）。
    *   将该图片重命名为 **`hi_qwq_ms.png`**。
    *   把 `hi_qwq_ms.png` 同样丢进上述的 `plugins` 文件夹中（也就是说，让图片和 `.dll` 放在同一个文件夹里）。
4.  配置完成后，直接正常启动游戏。你就会发现主界面的模组标志已经变成了你刚刚放入的精美缩小版专属图标！

---

## 📢 注意事项

*   请确保图片格式严格为 `.png`，且文件名必须全小写命名为 `hi_qwq_ms.png`。
*   模组自带的图片缩放逻辑已调节至最舒适的**小巧精致比例**，既能彰显个性，又绝对不会遮挡游戏自带的任何版本号或房间代码。
